# Wie man richtig Hilfe bekommt

##### Dieser Guide wird dir beibringen, wie man richtig Fragen im `realms-help`-Forum auf dem Discord Server stellt

### Einleitung

Wenn du eine Frage im `realms-help`-Forum stellst, ist es wichtig, möglichst viel Information zu liefern.

Dies wird es den Nutzern, die dir helfen wollen, um einiges einfacher machen, dein Anliegen zu verstehen. Somit können sie dir dann um einiges besser helfen.

### Bevor du eine Frage stellst

Manchmal ist es auch möglich, sich die Frage selbst zu beantworten, ohne, dass man einen neuen Post erstellt.

Daher solltest du vor dem Erstellen eines Posts die folgenden Dinge überprüfen:

1. Hast du im Internet nach einer Lösung gesucht?

    *(Damit ist auch das `realms-help`-Forum gemeint, nutz dazu das Suchfeature von Discord, um zu überprüfen, ob jemand das Problem schonmal hatte)*

2. Hast du in der Dokumentation von dem Plugin, das du verwendest, nach einer Lösung gesucht?

3. Hast du den Server Log nach Fehlermeldungen überprüft ([Erklärung hier](./reading-logs))?

### Die Frage stellen

Wenn du eine Frage im Forum erstellst, musst du die folgenden Dinge in deiner Nachricht angeben:

- Kontext, was passiert ist
- Logdateien vom Server
- Liste an Plugins auf deinem Realm
- Was du schon versucht hast, um das Problem zu lösen
- *Screenshots oder Videos (falls nötig, um das Problem zu beschreiben)*

Überprüfe deine Nachricht vor dem Absenden auf Rechtschreib- oder Grammatikfehler.
Man kann dir um einiges einfacher helfen, wenn deine Nachricht verständlich formuliert und ohne Rechtschreibfehler ist.

Befolge die [Regeln vom Discord Server](https://cytooxien.net/help/discord-rules).

### Dinge, die du vermeiden solltest

- Nicht auf Gegenfragen antworten, dies macht es für Nutzer um einiges schwieriger, dir zu helfen
- Das Aufteilen von Sätzen in mehrere Nachrichten (z.b.: Nachricht 1: `Aber`, Nachricht 2: `was`, Nachricht 3: `soll`, Nachricht 4: `ich`, Nachricht 5: `tun`, Nachricht 6: `?`)
- Ungeduldig sein, die Nutzer haben auch ein Leben und sie haben auch etwas anderes zu tun, als 24/7 Fragen zu beantworten
- Teammitlieder pingen, vor allem mehrere
- Das Erstellen von mehreren Posts oder das Cross-Posten der Frage in mehreren Kanälen
