# Eigenes Backup erstellen

##### Hier lernst du, wie du dein eigenes Backup erstellen kannst

Tägliche Backups reichen dir nicht aus? Oder möchtest du deine Realm herunterladen?


    1. Öffne das Web-Interface deines Realm.

    2. Gehe zum Bereich "Dateien".

    3. Wähle alle Dateien aus, indem du den Kasten links neben den Drei Punkten anklickst.

    4. Klicke auf den Button "Archivieren", dieser erscheint unten.

    5. Warte ein paar Augenblicke, während deine RealmDateien in ein tar.gz-Archiv gepackt werden.

    6. Sobald das Archiv erstellt wurde, klicke auf die 3 Punkte neben der Datei und klicke "Herunterladen".
