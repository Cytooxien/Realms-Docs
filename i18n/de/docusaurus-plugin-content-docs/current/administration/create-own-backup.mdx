# Eigenes Backup erstellen

##### Hier lernst du, wie du dein eigenes Backup erstellen kannst

Tägliche Backups reichen dir nicht aus? Oder möchtest du die Dateien von deinem Realm herunterladen?

    1. Öffne das Web-Interface deines Realms.

    2. Gehe zum Bereich "[Dateien](../web-interface/file-manager)".

    3. Wähle alle Dateien aus, indem du den Kasten links neben den drei Punkten anklickst.

    4. Klicke auf "Archivieren", dieser erscheint unten.

    5. Warte ein paar Augenblicke, während deine Realm Dateien in ein tar.gz-Archiv gepackt werden.

    6. Sobald das Archiv erstellt wurde, klicke auf die drei Punkte neben der Datei und klicke auf "Herunterladen".
