# Upload custom world

##### In dieser Guide erfährst du, wie du deine eigene Welt hochladen kannst

Du hast bereits eine eigene Welt? Kein Problem, der Guide zeigt dir Schritt für Schritt, wie du sie hochlädst.

      1. Öffne das Web-Interface von deinem Realm.

      2. Stoppe deinen Realm, falls der Server gerade läuft.

      3. Navigiere zum Bereich „Dateien“.

      4. Sende deinen Welt Ordner auf deinem PC zu ZIP.

      5. Ziehe die ZIP Datei per Drag & Drop in das Web-Interface oder klicke auf den Upload Button, um sie manuell hochzuladen.

      6. Nach dem Upload suchst du die datei, klicke auf die 3 Punkte daneben.

      6. Wähle „Entpacken“, und warte ein paar Sekunden, bis der Vorgang abgeschlossen ist.

      7. Merke dir den Namen des Ordners und starte dann deinen Realm.

      8. Gib den Befehl `/realm` ein, um das Interface zu öffnen.

      9. Klicke auf den Button „Welten“.

      10. Erstelle eine neue Welt, indem du oben links auf den grünen Button „Welt generieren“ klickst.

      11. Gib im Chat den Namen des Ordners ein, deine Welt wird automatisch importiert.
