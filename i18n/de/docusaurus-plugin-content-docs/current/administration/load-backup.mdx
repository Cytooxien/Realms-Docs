# Backup laden

##### So kannst du ein Backup von deinem Realm laden.

In den Cytooxien Realms musst du dir keine Sorgen um Datenverlust machen, es werden täglich automatische Backups erstellt.


**Backup laden, wenn du dich auf deinem Realm befindest:**

 1. Gib den Befehl `/realm` ein, um das [Realm Menu](../getting-started/settings) zu öffnen.

 2. Klicke unten rechts auf den „Einstellungen“ Button.

 3. Klicke auf den orangen Knopf „Realm wiederherstellen“.

Nun siehst du alle verfügbaren Backups deines Realms, je nach Boost-Level.
Der aktuell aktive Wiederherstellungspunkt wird hervorgehoben.

**Backup laden, wenn du in der Lobby bist:**

 1. Öffne den Navigator und teleportiere dich zu der [Realm-Lobby](../getting-started/realm-lobby).

 2. Gehe zum NPC „Meine Realms“.

 3. Linksklicke auf deinen Realm und wähle anschließend den „Einstellungen“ Knopf unten rechts aus.

 4. Klicke auf den orangen Knopf „Realm wiederherstellen“.

Wie zuvor siehst du alle verfügbaren Backups entsprechend dem Boost-Level.
Der aktuell aktive Wiederherstellungspunkt wird hervorgehoben.

 5. Klicke einfach auf das Backup, das du laden möchtest, dein Realm wird neu gestartet und lädt das ausgewählte Backup automatisch.
