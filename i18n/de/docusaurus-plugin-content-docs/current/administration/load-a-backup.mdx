# Backup laden

##### So kannst du ein Backup von deinem Realm laden.

In den Cytooxien Realms musst du dir keine Sorgen um Datenverlust machen, es werden täglich automatische Backups erstellt.


**Backup laden, wenn du dich auf deiner Realm befindest:**
    1. Gib den Befehl `/realm` ein, um das Web-Interface zu öffnen.

    2. Klicke unten rechts auf den „Einstellungen“ Button.

    3. Klicke auf den orangen Button „Realm wiederherstellen“.

    Nun siehst du alle verfügbaren Backups deines Realm, je nach Boost-Level.
    Der aktuell aktive Wiederherstellungspunkt wird hervorgehoben angezeigt.



**Backup laden, wenn du in der Lobby bist:**
    1. Öffne den Navigator und teleportiere dich zu den Realms.

    2. Geh zum NPC „Meine Realms“.

    3. Linksklicke auf deinen Realm und wähle anschließend den „Einstellungen“ Button unten rechts aus.

    4. Klicke auf den orangen Button „Realm wiederherstellen“.

    Wie zuvor siehst du alle verfügbaren Backups entsprechend dem Boost-Level.
    Der aktuelle Wiederherstellungspunkt wird dabei hervorgehoben.

5. Klicke einfach auf das Backup, das du laden möchtest, dein Realm wird neu gestartet und lädt das ausgewählte Backup automatisch.
