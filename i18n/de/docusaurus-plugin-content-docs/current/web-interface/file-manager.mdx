# Dateiverwaltung

##### Diese Seite erklärt, wie Dateien mit dem Web-Interface verwaltet werden.

Die Dateiverwaltung ist für das Verwalten von Dateien. Du kannst Dateien erstellen oder direkt hochladen mit den Knöpfen am oberen Rand oder per Drag-and-Drop.
Hier können auch bestehende Dateien bearbeitet werden. Im Editor kann zusätzlich eine Texthervorhebung eingestellt werden, die sich normalerweise an der Dateiendung orientiert.

Wenn du eine Datei in das vorherige Verzeichnis verschieben möchtest, nutze `../`. Zum Beispiel: `../ordner` ist der Ordner `ordner` im vorherigem Verzeichnis. 

Über die [Einstellungen](./other-tabs#settings) können Dateien mit SFTP bearbeitet werden.
